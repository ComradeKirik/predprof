<h1>Краткий экскурс по проекту для разработки</h1><br>
<b>app.py</b> - приложение проекта на Flask<br>
<b>DBoperations.py</b> - файл с операциями для работы с БД, создан для того, чтобы не захламлять app.py
<br><b>templates</b> - папка со страницами и темплейтами. Темплейты обязательно содержат в себе _template.<br>
<b>cleardb.sql</b> предназначена для очистки базы данных. Экзекутаете ее и БД очищается.
