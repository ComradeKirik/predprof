# Платформа для подготовки к олимпиадам

Программный комплекс, предназначенный для обучения, проведения состязаний и подготовки к олимпиадам по различным предметам. Проект реализован как современное веб-приложение с акцентом на интерактивность и доступность с любых устройств.

## Техническая архитектура

Система построена на базе **клиент-серверной архитектуры**, что обеспечивает:
* **Централизацию:** Данные хранятся и обрабатываются на сервере.
* **Безопасность:** Реализован строгий контроль полномочий. Доступ к данным предоставляется только авторизованным пользователям в соответствии с их ролями.
* **Масштабируемость:** Архитектура допускает вертикальное масштабирование при увеличении нагрузки.

Программная логика следует модели **MVC (Model-View-Controller)**. Это позволило разделить проект на три независимых уровня:
1. **Модель данных:** Отвечает за структуру информации и взаимодействие с базой данных.
2. **Визуальное представление:** Графический интерфейс пользователя.
3. **Контроллер:** Управляющая логика, связывающая интерфейс и данные.

## Технологический стек

### Backend (Серверная часть)
* **Язык программирования:** Python.
* **Веб-фреймворк:** Flask. Используется для обработки HTTP-запросов и реализации бизнес-логики.
* **Real-time взаимодействие:** Flask-SocketIO. Технология веб-сокетов позволяет обновлять данные на странице в реальном времени (например, в режиме соревнований) без её полной перезагрузки.

### База данных
* **СУБД:** PostgreSQL.
* **Модель хранения:** Stateful. Обеспечивает высокую скорость работы с реляционными данными и надежность их хранения.

### Frontend (Клиентская часть)
* **Технологии:** HTML5, CSS3, JavaScript.
* **Верстка:** Адаптивная блочная верстка (на `div`-контейнерах). Интерфейс автоматически подстраивается под любые разрешения экранов.
* **Совместимость:** Полнофункциональная работа на десктопных компьютерах, планшетах и смартфонах без установки дополнительных приложений.

## Основные функциональные возможности
* **Регистрация пользователей** Пользователь может зарегистрироваться под любым валидным адресом электронной почты.
* **Редактирование базы заданий** Администратор может наполнять базу заданий, в том числе с использованием импорта из системы sdamgia.ru. Администратор может выполнять экспорт заданий в JSON-формате для их последующего импорта в другой экземпляр системы.
* **Режим тренировки** Позволяет выполнять задания, в том числе с использованием подсказок, и отслеживать свой прогресс.
* **Интерактивные соревнования:** Реализация активного режима состязаний с мгновенной обратной связью через веб-сокеты.
* **Управление доступом:** Встроенная система авторизации и разграничения прав.
* **Интеграция с внешними ресурсами:** Предусмотрена возможность импорта заданий из сторонних систем (например, sdamgia.ru).

## Порядок установки
* Установить Python 3.13+
* Установить дополнительные модули Python: 
    + pip>=25.2
    + typing_extensions>=4.15.0
    + attrs>=25.4.0
    + Flask>=3.1.2
    + python-dotenv>=1.2.1
    + Werkzeug>=3.1.3
    + psycopg2>=2.9.11
    + bcrypt>=5.0.0
    + flask_socketio>=5.6.0
    + pytest>=8.0.0
    + pytest-mock>=3.12.0
    + pytest-flask>=1.3.0
    + coverage>=7.0.0
    + openai>=2.16.0
* Установить PostgreSQL версии 18+
* Создать пользователя postgres и базу данных players
* Клонировать репозиторий с программным кодом
* Настроить конфигурационный файл .env с переменными окружения для подключения к базе данных на основе шаблона example.env, в котором:
    + SECRET_KEY - это секретная строка, используемая для подписи криптографических данных в сессии пользователя (cookies)
    + PG_PASSWORD - это пароль пользователя от базы данных PostgreSQL
* Запустить программу с помощью команды `python3.13.exe app.py`
* Для прерывания работы приложения можно использовать сочетание горячих клавиш `Ctrl+C`
